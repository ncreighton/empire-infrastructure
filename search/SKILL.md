# Search

Infrastructure component for the Empire ecosystem. Cross-project search engine that finds code, knowledge base entries, configs, manifests, and deprecated patterns across the entire empire in milliseconds.

## Key Functions

- `full_search(hub_path, query, mode)` — Run full search across all categories (code, knowledge, config, deprecated, manifest)
- `search_code(hub_path, query_terms)` — Search code files across all projects with canonical source boosting
- `search_knowledge(hub_path, query_terms)` — Search knowledge base markdown entries
- `search_configs(hub_path, query_terms)` — Search shared configuration and schema files
- `search_deprecated(hub_path, query_terms)` — Search deprecated methods and blacklist entries
- `search_manifests(hub_path, query_terms)` — Search project manifest files
- `score_match(query_terms, text, boost)` — Score how well text matches query terms with word boundary and position bonuses

## Search Modes

| Mode | Flag | What It Searches |
|------|------|-----------------|
| All | (default) | Everything |
| Code | `--code` | Source files (.js, .ts, .py, .php, .md, .json, etc.) |
| Knowledge | `--kb` | Knowledge base markdown entries |
| Config | `--config` | Shared configs and system schemas |
| Deprecated | `--deprecated` | BLACKLIST.md and migration docs |
| Manifest | `--manifest` | Project manifest files |

## CLI Usage

python search.py --kb "rate limit"               # Knowledge base only
python search.py --config "image quality"         # Configs only
python search.py --deprecated "publishPost"       # Deprecated items only
