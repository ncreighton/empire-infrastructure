## FORGE + AMPLIFY Intelligence System (INSTALLED)

- **Source**: velvetveil-printables / CLAUDE.md
- **Confidence**: 0.8

This project has a fully customized FORGE+AMPLIFY intelligence system at `intelligence/`.

### Architecture

```
intelligence/
├── __init__.py                 # Central imports
├── master_orchestrator.py      # Brain: connects all systems
├── forge_pdf_engine.py         # FORGE: 5 intelligence modules
├── amplify_pdf_pipeline.py     # AMPLIFY: 6-stage enhancement pipeline
├── super_prompt_engine.py      # Maximizes every prompt
├── pdf_qa_engine.py            # Scores PDFs, auto-generates fixes
└── asset_forge.py              # Multi-API asset generator
```

### Quick Commands

```python
from intelligence import ForgePDFIntelligence, AmplifyPDFPipeline, SuperPromptEngine, PDFQAEngine, AssetForge
from intelligence.master_orchestrator import MasterOrchestrator

#### FULL PIPELINE: Natural language → production-ready spec
orchestrator = MasterOrchestrator()
result = orchestrator.create("Samhain Ritual Kit")
print(result.summary)

#### FORGE: Pre-generation intelligence
forge = ForgePDFIntelligence()
intel = forge.pre_generation_analysis(
    product_config={"type": "sabbat_kit"},
    content_prompt="Write a Samhain kit",
    image_prompt="Mystical altar"
)

#### AMPLIFY: Enhance generation request
amplify = AmplifyPDFPipeline()
result = amplify.amplify_quick("sabbat_kit", "Samhain Ritual Kit")

#### SUPER PROMPTS: Enhance any prompt
prompts = SuperPromptEngine()
enhanced = prompts.enhance("Create samhain content", prompt_type="content",
                           product_type="sabbat_kit", sabbat="samhain")

#### QA: Score a PDF
qa = PDFQAEngine()
report = qa.score_file("output/samhain_kit.pdf", "sabbat_kit")
fix_css = qa.generate_fix_css(report)

#### ASSETS: Plan and cost estimate
assets = AssetForge()
estimate = assets.estimate_cost("sabbat_kit", "samhain")
```

### FORGE Modules
| Module | Purpose |
|--------|---------|
| **SCOUT** | Gap detection -- finds missing elements in product configs |
| **SENTINEL** | Prompt optimizer -- scores and auto-enhances all prompts |
| **ORACLE** | Pattern predictor -- predicts issues + seasonal intelligence |
| **SMITH** | Auto-fix generator -- 18 fix templates for PDF/typography issues |
| **CODEX** | Learning engine -- SQLite memory that improves with every generation |

### AMPLIFY Stages
| Stage | Purpose |
|-------|---------|
| **ENRICH** | Pulls brand data, niche intelligence, seasonal context |
| **EXPAND** | Adds bonus sections, format variants, audience segments |
| **FORTIFY** | Font fallbacks, image validation, retry logic, safety checks |
| **ANTICIPATE** | Predicts typography/layout/print issues, generates preventive CSS |
| **OPTIMIZE** | Routes to best API for each task, balances cost vs quality |
| **VALIDATE** | Final completeness check before generation |

### AMPLIFY Mode Active
- Every prompt is auto-enhanced through SuperPromptEngine
- Every generation is pre-analyzed by FORGE
- Every output is scored by PDFQAEngine
- Every result feeds back to CODEX for learning
- Bold, decisive execution -- no permission asking

### Before Any Task
1. Run `orchestrator.analyze(request)` for full intelligence briefing
2. Run `orchestrator.create(request)` for complete generation spec
3. Use enhanced prompts from `result.enhanced_prompts`
4. Apply preventive CSS from `result.amplify_result["preventive_css"]`
5. Score output with `orchestrator.score_output(file_path, product_type)`
