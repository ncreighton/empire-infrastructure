# Nexus

Infrastructure component for the Empire ecosystem. Provides git hooks and n8n workflow integration for Project Mesh cross-project coordination.

## Key Functions

- `pre_commit()` — Block deprecated patterns in staged files before commit
- `post_commit()` — Trigger sync notification if shared-core files changed, update manifest timestamps
- `pre_push()` — Full manifest validation before push
- `install_hooks(project_path)` — Install pre-commit, post-commit, and pre-push hooks into a project's .git/hooks/

## How It Works

1. **Pre-commit**: Scans staged files against deprecated code patterns from the hub's BLACKLIST. High-severity violations block the commit.
2. **Post-commit**: Detects shared-core changes and suggests sync. Updates `last_human_touch` in the project's manifest.
3. **Pre-push**: Validates all manifest JSON files before allowing push.
