## Key Insights from Video Transcripts

### From SEO Blog Writer Training (Live)
- Global background limited to ~1,000-1,200 words
- Up to 3 URLs can be scraped for global background
- Background is for **knowledge the AI doesn't have**, NOT for prompts/instructions
- Over 5 H2s with global background risks repetition -- use subheading backgrounds too
- Different AI models for outline generation vs article writing is recommended
- Custom prompts insert between initial generation and skinny paragraphs/translation
- Automatic keywords give AI new directions (not volumetric SEO keywords)
- Manual keywords: up to 150, control density per subsection

### From Web Scraping Basics
- Magic commands: one URL at a time, use "summarize=" prefix for summary-only
- Scraped summaries stored in clipboard for reuse
- ScrapeOwl required for Amazon/Google (rotating proxy)
- Amazon gets special CSS class extraction (product-specific data only)
- Non-Amazon sites may include extraneous footer/sidebar data -- review summaries
- Scraping costs ~$0.002 per 10,000-word article (turbo)

### From Custom Outlines
- H2 = plain line, H3 = one dash prefix, H4 = two dashes prefix
- Variables: {list}, {table}, {img}, {yt}, {model=name}, {voice_2p}, {cta}, {cp_name}
- Copy/paste outlines from clipboard with option 1
- Can save outlines for reuse

### From Product Reviews
- Trigger product layout: 2 paragraphs + pros/cons list
- Tail-end prompt customizable for product specs output
- ScrapeOwl $5/month for 10,000 credits
- Review scraped summaries before writing, especially non-Amazon sites
